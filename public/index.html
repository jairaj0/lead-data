📋 JSON format

Any kind of lead works — doctor, hotel, shop, advocate, hospital, anything. The JSON must be an array ([ ... ]), with each lead an object. Only name is required — everything else is optional. Alternate key names (like Name, Phone, mobile, category) also work — the app converts them automatically.
